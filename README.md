## 概要

この Web アプリで以下のことができます。

- チームメンバーの練習目標・結果が共有できる
- 練習動画のリンクを保存できる
- 次回練習部の天気と気温から、持っていくべき水分量を提案

**Available for both PC device 　 and smartPhone.**
